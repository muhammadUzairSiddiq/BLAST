# This Is Blast (Prototype)

A **practice prototype** inspired by *This Is Blast* by Voodoo, built using **Unity** to explore **hyper-casual gameplay mechanics**, **reactive systems**, and **clean modular architecture**.

---

## 🎮 Overview

**This Is Blast (Prototype)** focuses on the core mechanics of dynamic cube interactions, destruction sequences, and responsive movement.
It was developed as a **technical and design exercise** to simulate the visual feel, pacing, and interaction design of modern hyper-casual titles.

---

## ✨ Core Features

* **Reactive Cube Destruction System**
  Cubes detect collisions, play feedback animations, and trigger column movement updates dynamically.

* **Column-Based Position Mapping**
  Ensures precise positional updates during gameplay without floating-point drift.

* **Seamless Feedback System**
  Integrated movement and rotation feedback for impactful interactions.

* **Modular Architecture**
  Clean separation between managers (`CubeManager`, `LevelManager`, `ColorTransitionSystem`) for easy scalability and debugging.
